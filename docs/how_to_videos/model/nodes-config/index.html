Node Configurations

#Edit & Configure Node

Start designing your model by editing and configuring Nodes.

#Configuration

#Data Source

Assign primary value to your node from the series of values coming out of your Data Source

#Mathematical Operands

Use mathematical operands for quick calculation derived from Addition, Subtraction, Division & Multiplication

#Formulas

Leverage spreadsheet like formulas & static identifiers to calculate node value (no coding or programming required)

#Time Aggregation

Choose options to aggregate values over time periods based on the selected time aggregation method

#Linked Node

Learn what is a Linked Node and how to use it for instant visualization

#Desired Trend

Users can opt for a desired trend to visualize the model and variance impact

#Inserting Manual Values

Primary values can be inserted in node using this method

#Constant Node

Allows a user to configure constant primary value throughout the time period

#Display Options

Decide whether you want to show the node as a derived one or simply just hide it from the tree widget.

Choose primary value scaling, decimal points, prefix and suffix.

#Composite Nodes

Bring in any two important nodes and make them a part of your primary node

#Simulation

Simply allows realtime change to node values and its impact on other connected nodes and KPIs

#Business Definition Text in Node

Add texts like description, headers & footers for node pop-up display

#Conditional Formatting

Learn Conditional Formatting in ValQ by customizing your variance threshold limits and setting rules

#Customize Secondary KPIs

Allow nodes to display additional customized KPIs which could be used as secondary values or measures for insightful analysis
