Create New Model

In the Model tab, click New to start building a new model with the options shown in the screenshot below:

Create a new model

  1. Open a Sample Model - View a built-in sample ValQ model
  2. Create a Simple Model - ValQ builds a simple model from the data you import
  3. Create an Advanced Model - Build a ValQ model from scratch using the data you import
  4. Import an Advanced Model - Advanced users can import an already built existing model
note

Before starting to build a ValQ model, the first time users can click the below the icon in the nav-panel to learn more about the ValQ and watch the tutorials.

1. Open a Sample Model#

The ValQ has 4 in-built model samples in the Open a Sample Model option. Select any of the samples to explore ValQ and it's business cycles.

Sample models

Below is a screenshot that shows sample model data for the "Mining Industry".

Mining Industry sample

2. Create a Simple Model#

To build a Simple Model, first you need to import data into Power BI and activate ValQ. This is followed by creating a simple model on ValQ.

The steps to create a Simple Model has been covered under Using ValQ - An Example or you can refer this pdf for step by step instructions on how to build a Simple Model.

A Simple model looks like this

Simple model

Note
The Calculation Method for a Simple Model is always set to Sum Children

2.1 Edit a simple model#

The nodes on a Simple Model are disabled and you cannot perform any configuration on it. But you can generate a child node of the top node and proceed to configure it based on your needs.

Edit a simple model

2.2 Convert to advanced model#

To configure the nodes of a Simple Model, convert the Simple Model to Advanced using the magic wand icon highlighted below.

Magic wand

Click on the icon and you will be prompted with the Message window as shown below.

Convert to advanced model

After clicking Yes, you have successfully converted the Simple Model into an Advanced Model. You will now be able to configure all the node levels similar to an Advanced Model.

Advanced model

3. Create an Advanced Model#

Use Create an Advanced Model option to build a model on ValQ from scratch manually.

Create an Advanced Model

In the below screen, a parent node and two child nodes have been added by clicking the + button. The node levels are displayed in the Node Editor pane as shown in the below Figure.

Advanced model

Now you can go ahead and build the model as per your needs using the Model tab options.

4. Import an Advanced Model#

Import from an Export File

With Import from an Export File option, you will be able to paste the Export file data format from an already exported tree as shown in the below Figure. This Export File will be generated by navigating to the Settings tab and clicking the Export button like this.

Import a Tree

On clicking Finish, you will be able to view the model in the ValQ screen as below.

Advanced model
