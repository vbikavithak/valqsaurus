How to install ValQ?

In this section, you will:

  • Learn how to install ValQ on Power BI Desktop
  • Launch ValQ

Get ValQ#

You can use any of the following three ways to get ValQ added to Microsoft Power BI:

  1. Import a visual from a file
  2. Get more visuals - AppSource
  3. Get more visuals - My organization

1. Import a visual from a file#

Import a visual from a file option is used when you have ValQ as a .pbiviz file in your local machine.

1. a) To get started, click on the ellipses under the Visualizations pane and choose Import a visual from a file option as shown in the image below.


  1. b) Now select the .pbiviz file available in your local desktop to import ValQ into Power BI.

  1. c) Once the file gets imported, you will see the message as shown below. Click OK.

  1. d) You have successfully added ValQ to Microsoft Power BI. You will now be able to see ValQ now under the Visualizations pane.

2. Get more visuals - AppSource#

Get more visuals option can be used to download ValQ from the AppSource marketplace.

2. a) To get started, click on the ellipses under the Visualizations pane and choose Get more visuals option.


  1. b) If you have not signed in to Microsoft Power BI already, you would be prompted to sign in to access the marketplace.

  1. c) Simply search for ValQ in the search bar and add the visual to Power BI by clicking Add button.

  1. d) Once the file gets imported, you will see the message as shown below. Click OK.

  1. e) You have successfully added ValQ to Microsoft Power BI. You will now be able to see ValQ now under the Visualizations pane.

3. Get more visuals - My organization#

Get more visuals option can also be used to download ValQ when the visual is a part of My organization marketplace.

3. a) To get started, click on the ellipses under the Visualizations pane and choose Get more visuals option.


  1. b) If you have not signed in to Microsoft Power BI already, you would be prompted to sign in to access the marketplace.

  1. c) Switch to My organization tab to identify the ValQ visual and click Add to get ValQ.

Note
ValQ will be available here only if the administrator from your organization has enabled ValQ


  1. d) Once the file gets imported, you will see the message as shown below. Click OK.

  1. e) You have successfully added ValQ to Microsoft Power BI. You will now be able to see ValQ now under the Visualizations pane.

Launch ValQ#

  • To begin, click on the ValQ icon. You will see ValQ visual on the canvas.

  • Expand the visual to fit the screen. You can now start using ValQ by assigning value to the visual.
