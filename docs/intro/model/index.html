Using ValQ - An Example

Learn how to create your first visualization using ValQ in less than 5 minutes. Try it out now with the below resource(s):

#Step-by-Step Guide

In this section, you will learn how to visualize sales data using ValQ in less than 5 minutes. To do this you will have to

  1. Import data from Microsoft Excel (or any other data source)
  2. Assign your data fields to ValQ

#1. Import data from Excel

  1. a) We have a simple dataset as shown below that measures sales performance for a combination or region, product category & sales rep. Download the dataset and try here .

  1. b) To import this data into Power BI, go to File and select Get Data. Now choose Excel to import the sales data.

  1. c) Go to the file location and select the file to be imported. Click Open.

  1. d) Select the Sales Data(Columnar) checkbox to preview the dataset and click Load.

  1. e) You will now see the dataset under the Fields pane on the right.
