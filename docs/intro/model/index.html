Using ValQ - An Example

Learn how to create your first visualization using ValQ in less than 5 minutes. Try it out now with the below resource(s):

Step-by-Step Guide#

Learn how to visualize your sales data with the following steps:

  1. Import data from Excel
  2. Assign your data fields to ValQ

1. Import data from Excel#

  1. a) We have a simple dataset as shown below that measures sales performance for a combination or region, product category & sales rep. Download the dataset here. and try for yourself.

  1. b) To import this data into Power BI, go to File and select Get Data. Now choose Excel to import the sales data.

  1. c) Go to the file location and select the file to be imported. Click Open.

  1. d) Select the Sales Data(Columnar) checkbox to preview the dataset and click Load.

  1. e) You will now see the dataset under the Fields pane on the right.

1. f) Click on the ValQ icon under the Visualizations pane. If you donâ€™t see this option, you will have to first download and install ValQ . Now resize the ValQ visual in the canvas to full screen.


1. g) Assign Sales Forecast from the Fields pane to the visual to activate ValQ. You will now see the following. Close the popup window.


Note
At least one Value field has to be added to run ValQ

2. Assign your data fields to ValQ#

  1. a) Click on the Create a simple model option.

  1. b) You will only see Sales Forecast now. We need to start assigning additional fields now.

  1. c) Map additional fields to the visual as shown below.

  1. d) Note how ValQ has instantly built the visualization from the data.

  1. e) Click on the > icon at the right of the East node to reveal its children. You can expand/collapse any node like this.

You have built your first visualization using ValQ. Take some time to explore this model further.
