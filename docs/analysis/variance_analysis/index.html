Variance Analysis

Variance Analysis calculates variances against a benchmark for the KPIs to understand how it's performance impacts the business.

ValQ's Variance Analysis helps compare the values of two or more data series for each of the nodes in the model and calculate variances - absolute and percentage for all the nodes.

#Pre-requisite

A ValQ model with Comaprison is needed to perform Variance Analysis

#Access Variance Analysis

To access Variance Analysis, click on the Variance Analysis option from the Analyze tab drop-down as shown below


Here's the Variance Analysis screen on ValQ


Now let's take a look at the contents of Variance Analysis and learn how to build a Variance Analysis report.

#Features of Variance Analysis

The Variance Analysis screen consists of two panels:

On top of the page, you will find a panel containing the features that help to configure the analysis report as per your needs.


The other panel shows the Variance Analysis report prepared based on the model that you have built and it's configuration.


Click on any option to know more

2Choose Compared Series
3Set period for variance analysis at period level
4Choose Time Range for variance analysis

#1. Set Base Series

Base Series can be selected either from the available Data Series or Scenarios created under the Simulate tab.

Here, a Data Series - '2020 Actuals' is set as Base Series


Instead, a Scenario that you have created in the Simulate tab can be set as Base Series.

Here, a Scenario - 'Scenario 1' is set as Base Series

#2. Choose Compared Series

One or more Data Series can be choosen as Comaprison Series as shown below.

Here, Data Series - '2020 Actuals' and '2020 Forecast' are choosen as Compared Series

#3. Select period for Variance Analysis at period level

The period for which you would like to perform Variance Analysis can be set as described below

#Same Period for Base Series and Compared Series

Select a period from the drop-down to get the same period values of both the series. In the below example, 'Apr' is selected from the drop-down. Variance Analysis is done for April period values of 2020 Actuals and April period values of 2020 Forecast.

#Custom Period for Base Series and Compared Series

To customize your selection, click on the Custom option from the drop-down and set different periods for Base Series and Compared Series.

#4. Choose Time Range for Variance Analysis

#Built-in Time Range

Choose the duration from the drop-down to specify time range for which you would like to run the analysis. Here, you can select multiple time range like Full year, Quarter, Year-to-date, etc. ValQ performs Variance Analysis for each of the selected range.


The time range - Periodic, Quarter, Full Year options have been selected as highlighted in the above image. The canvas displays the Variance Analysis report as below based on our selection. Analysis for Full Year can be seen as you slide on the canvas.

#Add Custom Time Range

You can also add a Custom time range by selecting the custom option from the drop-down. The Custom time range can be selected along with other time range and it is displayed at the end of the report.

Hover over the selection period(s) box to select the start month and the end month.

OptionDescription
+AddAdd a new Custom time range
ClearClear the selection to make a new selection
ApplySaves the selection as a Custom time range

Check the box to add the Custom selection to the report.


The Custom time range - Jan-Apr has been set as shown in the above image. The canvas displays the Variance Analysis for the Custom time range as below

#5. Absolute Variance and % Variance

Check the Absolute variance box to display variances in absolute terms.
Check the % variance box to add variances as percentage to the report.

ValQ adds the variances to all the nodes on the model and for the selected time range. The positive variances are shown in green and the negative ones in red.

#6. Show/ Hide Unit column

The unit column is seen by default on Variance Anaylsis as below. To hide the column, uncheck unit to disable the feature.

#7. Download as Excel

Download as Excel is used to download the Variance Analysis report as an excel file to your system.


Note
Download as Excel feature is available only on Power BI Service.

#8. Show Hidden Nodes

Enable Show hidden nodes option to display the nodes that were hidden while building the model under the Model tab.

Here, the node 'East' and its children are hidden


Check the Show hidden node option to display the 'East' node and its children. Note that the Overall Result remains same.


Note
On enabling Show hidden nodes option, the nodes stays hidden on the other tabs.

#9. Search for a node

Search for a particular node by its title as highlighted below. This feature can be used to identify a node when a model has a large number of nodes.

#10. Apply Filter to report

Use the Filter option to analyze critical influencers.

There are two types of filters:

Filter by Parent: This allows you to analyze variance for a specific branch within the model. For example, you may be interested in analyzing variance for a particular location or product.

Filter by Variance: This allows you to identify top ânâ analysis for categories showing the most variances for a particular period chosen from the drop-down.

Note
You have the option to select whether you want to display only positive variances or only negative variances or both.

In this example, filter By Variance has been applied to filter the top 5 variances for Full Year.


The below image shows the Variance Anaysis for top 5 variances for the full year. Observe the results to see both positive and negative variances. The Value Bar displays the filter that has been applied.

To remove the filter simply click on the 'X' next to the filter.

Here's the same filter with only negative variances. The report below shows the bottom 5 performers.

#11. Hide the features panel

Click on '^' to hide the features panel on the Variance Analysis screen. Hide the panel to take a clear look at the report and you can use this view as presentation mode.
